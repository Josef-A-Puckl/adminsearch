Great module proudly presented by [OXID Hackathon 2017](https://openspacer.org/12-oxid-community/185-oxid-hackathon-nuernberg-2017/) ;-)

Features

	Global admin search for
	- articles
	- categories
	- cms pages
	- users
	- orders
	- vendors
	- manufacturers
	- modules

Installation

```
composer require oxid-community/adminsearch
```
